# UI/UX Documentation

Tài liệu giao diện và trải nghiệm người dùng của hệ thống OpenAcademix.

## Design System

Hệ thống sử dụng Material-UI (MUI) làm design system chính.

### Components
- Buttons, Forms, Tables
- Dialogs, Drawers, Navigation
- Cards, Lists, Typography

### Theme
- Primary color: Blue
- Secondary color: Gray
- Typography: Inter font family

## Component Library

Các component tùy chỉnh:
- `DocumentViewer` - Xem tài liệu
- `MarkdownViewer` - Hiển thị markdown
- `PDFViewer` - Hiển thị PDF
- Navigation sidebars cho từng module

## Responsive Design

Hệ thống hỗ trợ:
- Desktop (1920px+)
- Tablet (768px - 1919px)
- Mobile (320px - 767px)

## Accessibility

- WCAG 2.1 Level AA compliance
- Keyboard navigation support
- Screen reader friendly
- High contrast mode support
