# Modules

Tài liệu chi tiết cho từng module của hệ thống OpenAcademix.

## Danh sách modules

### Module Quản lý tổ chức
Quản lý cấu trúc tổ chức, đơn vị, và quan hệ giữa các đơn vị.

**Tính năng chính:**
- Quản lý đơn vị tổ chức (Org Units)
- Quan hệ giữa các đơn vị
- Phân quyền theo đơn vị
- Lịch sử thay đổi

### Module Quản lý nhân sự (HR)
Quản lý thông tin nhân viên, hợp đồng, và đánh giá hiệu suất.

**Tính năng chính:**
- Quản lý nhân viên
- Hợp đồng lao động
- Đánh giá hiệu suất
- Phân công công việc

### Module Chương trình đào tạo (TMS)
Quản lý chương trình đào tạo, học phần, và đề cương chi tiết.

**Tính năng chính:**
- Quản lý học phần
- Chương trình đào tạo
- Đề cương chi tiết (Syllabus)
- Phiên bản học phần

### Module Học phí (Finance)
Quản lý học phí, biểu giá, và thanh toán.

**Tính năng chính:**
- Biểu giá học phí
- Quy tắc tính học phí
- Hóa đơn và thanh toán

### Module Tuyển sinh
Quản lý quá trình tuyển sinh và nhập học.

**Tính năng chính:**
- Quản lý đợt tuyển sinh
- Hồ sơ ứng viên
- Quy trình xét tuyển

### Module Đào tạo
Quản lý quá trình đào tạo và học tập.

**Tính năng chính:**
- Lớp học phần
- Đăng ký học
- Điểm số và kết quả học tập
