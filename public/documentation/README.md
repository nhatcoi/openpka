# Tài liệu hệ thống

Thư mục này chứa các tài liệu của hệ thống OpenPKA.

## Định dạng được hỗ trợ

- **Markdown** (.md, .markdown): Tài liệu markdown
- **PDF** (.pdf): Tài liệu PDF

## Cách sử dụng

1. Đặt các file tài liệu vào thư mục này
2. Truy cập trang `/documentation` để xem tài liệu
3. Chọn tài liệu từ danh sách để xem nội dung

## Cấu trúc thư mục

```
public/documentation/
├── README.md
├── N01_G02_TLCaidat.md
└── ... (các file khác)
```

## Lưu ý

- File markdown sẽ được render với syntax highlighting cho code blocks
- File PDF có thể zoom in/out và điều hướng trang
- Tất cả các file trong thư mục này sẽ được tự động liệt kê trong trang documentation
